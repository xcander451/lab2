# Лабораторная работа №2: Обнаружение и обработка выбросов

## Описание
Данная лабораторная работа посвящена методам обнаружения выбросов (аномалий) в данных. В рамках работы реализованы статистические методы, проверка нормальности распределения, а также методы на основе локальной плотности.

## Цель работы
Освоить методы обнаружения выбросов на основе статистических критериев

Научиться проверять гипотезу о нормальности распределения

Реализовать методы Z-оценки и межквартильного расстояния (IQR)

Применить модельные и метрические методы обнаружения аномалий

Сформировать обоснованные решения по обработке выбросов с учетом предметной области
